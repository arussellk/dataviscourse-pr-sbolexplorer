SBOLExplorer

SBOLExplorer - The Synthetic Biology Open Language Explorer

SBOLExplorer is a visualization of genetic components using the Synthetic Biology Open Language (SBOL). SBOL defines a data format that represents synthetic genetic parts that researchers can use as DNA building blocks to create genetic constructs or larger synthetic genetic components. SBOLExplorer visualizes this dataset of synthetic genetic parts as well as the inheritance relationships between them.

Data

Our data is sourced from SynBioHub. Through SynBioHub, users can view data about a genetic component by visiting its reference page (BBa_I5610 example reference). Notice that SynBioHub does not have a way to view the hierachical nature of a genetic component tree. SBOLExplorer was created to fill this need.
