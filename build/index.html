Data

Our data is sourced from SynBioHub. Through SynBioHub, users can view data about a genetic component by visiting its reference page (BBa_I5610 example reference). Notice that SynBioHub does not have a way to view multiple layers of a genetic component tree. SBOLExplorer was created to fill this need.
